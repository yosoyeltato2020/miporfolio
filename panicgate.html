PanicGate
Alarma silenciosa, camuflada como una calculadora

Protege a las personas en situaciones de emergencia.
PanicGate parece una calculadora normal, pero permite activar una alarma silenciosa introduciendo un código secreto.
¡Innovación en seguridad, ideal para oficinas, tiendas o cualquier lugar!
Solicita demo

¿Por qué PanicGate?

Aspecto inocente
Simula ser una calculadora, pasando totalmente desapercibida ante un atacante.
Códigos secretos
Activa acciones de seguridad (bloqueo, alerta, registro, etc.) solo con el código correcto (*911*, *112*).
100% silenciosa
No muestra señales visibles ni emite sonidos al activar la alarma.
Modular y ampliable
Permite añadir nuevas acciones de seguridad solo con módulos Python adicionales.
Registro y evidencia
Todas las activaciones quedan registradas en un archivo de log para auditoría.

¿Cómo funciona?

1. Calculadora normal
Se abre como una simple app de calculadora.
2. Código secreto
El usuario autorizado introduce un código como *911*.
3. Activación silenciosa
Se ejecuta la acción preconfigurada, sin alertar al atacante (bloqueo, registro, alerta, etc.).
4. Registro seguro
Toda acción queda en un archivo log, sin notificar al atacante.
